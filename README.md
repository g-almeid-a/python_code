📚 Projetos em Python — Prática de Fundamentos

Este repositório reúne uma coleção de pequenos projetos desenvolvidos em Python com o objetivo de praticar conceitos fundamentais da linguagem.

Cada arquivo representa um exercício ou mini-projeto focado em um tema específico.

🚀 Objetivo

Praticar lógica de programação e consolidar conhecimentos básicos de Python, como:

Estruturas condicionais (if, else)
Estruturas de repetição
Manipulação de listas
Entrada e saída de dados
Geração de valores aleatórios

🧠 Projetos incluídos

📋 Manipulação de Lista

Criação e modificação de listas
Iteração com loops

🔐 Gerador de Senhas

Geração de senhas aleatórias
Uso da biblioteca random

🎲 Sorteio

Seleção aleatória de elementos
Simulação de sorteios simples

🔀 Condicionais (if/else)

Estruturas de decisão
Validação de entradas

▶️ Como executar

Clone o repositório:
git clone https://github.com/seu-usuario/seu-repositorio.git

Entre na pasta:
cd seu-repositorio

Execute qualquer arquivo:
python nome_do_arquivo.py

💡 O que aprendi

*  Organização de código em arquivos separados

*  Criação de funções

*  Lógica de programação

*  Bas práticas básicas em Python
